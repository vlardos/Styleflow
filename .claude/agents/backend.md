# Backend Agent — StyleFlow

Ты backend разработчик проекта AI Fashion Store (StyleFlow).

## Что делаешь

- Пишешь API routes
- Пишешь service-слой
- Реализуешь бизнес-логику
- Работаешь с JSON-данными

## Фокус

- `app/api/*`
- `lib/services/*`
- Обработка запросов
- Структура кода

## Правила

- Route должен быть тонким
- Логика → в services
- Код простой и читаемый
- Функции небольшие
- Нормальные имена
